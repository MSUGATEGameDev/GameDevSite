About This Program

This class is an intro to game design using Unity (a free full-featured game engine).
We will also touch on C# coding in Visual Studio (an IDE packaged free with Unity)
and 3D model design in Blender (an open-source and completely free 3D design studio).

While time restrictions in-class will not provide time to get in-depth on all aspects of game design, we have filled this site with resources to help you on your continued game design journey.

Please Note: This site contains links to 3rd party videos on YouTube. While the creators of these videos are highly professional, a few of their videos reference games which are not appropriate for all ages.

Setup & Guides

Here are some guides to recap what we've discussed in class and expand upon themes we weren't able to go over in-detail.

  • Setting Up Your Environment

    This page will walk you through installing the main software we need for 3D game development: Unity, Visual Studio, Blender, and GitHub.

    It will also show you how to obtain your project files to continue your work from class.

  • Working in Unity Editor

    At some point, we will nicely write out our in-class instructions for navigating the Unity Editor.

    In the meantime, here is an excellent video by Game Maker's Toolkit on YouTube that outlines all the basics of using Unity.

  • Using GitHub

    This video by Kevin Stratvert on YouTube provides and in-depth look at the backup/version control standard Git, and the popular website known as GitHub, which are are using in our class.

    There are also basic GitHub instructions on our Environment Setup Page

  • Building Your Game

    This page will walk you through the game build, which is the process of converting your game from a project in the Unity editor to an executable or install file for others to enjoy.

  • Obtaining Pre-Made Assets

    The Unity Asset Store has a large collection of models, templates, and game features ready to import directly into your game.

    itch.io is an open storefront that allows people to post their games and ready-to-go game assets. Note that assets are not guaranteed to be compatible with Unity and may require conversion.

  • Creating 2D Assets

    We highly recommend Krita as the best all-around application for making 2D art. Their official channel on YouTube has some great tutorials.

    Inkscape is another great tool for 2D vector art. (Art that is based on line positions, not pixels, so it scales well to all sizes.)

  • Creating 3D Assets

    Blender Guru on YouTube has a great tutorial that walks you through the basics of Blender by making a donut.

Additional Guides

Here are some guides to expand your game design knowledge beyond what we had time to cover in-class.

  • 2D Game Development

    Here is series by Brackeys on YouTube about creating 2D games in Unity.

    The video is a little old so Unity has changed a bit since then, but most of the essential parts are still relevant.

  • 3D Game Development

    Here is series by Brackeys on YouTube about creating 3D games in Unity.

    The video is a little old so Unity has changed a bit since then, but most of the essential parts are still relevant.

  • VR Development

    YouTuber ValemTutorials has a step-by-step guide to make a Meta Quest or PC VR game in Unity.

  • Coding

    YouTuber Brackeys has an excellent beginners tutorial on coding in C# (the language used in Unity and Godot).

    YouTuber Infallible Code has a lot of great videos about various best-practices in coding, which help you to write efficient and reliable code.

  • Game Design Theory

    YouTuber Game Makers ToolKit's whole channel talks about game design ideas without going into technical details. It's a great place to learn about game design when you don't have the focus to work on concrete skills.

    Masahiro Sakurai (the Creator of Super Smash Bros. and Kirby) has a YouTube channel that offers insightful and easy-to-understand advice about game development. (Japanese with English Subtitles)

    YouTuber Brackeys has a series that covers topics about how to achieve a truly enjoyable game by balancing Difficulty, Storytelling, Appearance, ...etc.

  • Other Game Engines

    Brackeys on YouTube has some newer videos walking through the basics of creating games in Godot (a full-featured, completely free game engine).

    Matt Aspland on YouTube has a ton of videos walking through many aspects of building a game in Unreal Engine.

  • Project Management

    You will want to make sure you have a way to jot down your thoughts, organize them into checklists that need to be done, and have a way to track and collaborate.

    We'd recommend you try a website called Trello, which uses cards, boards, and checklists as an easy project management solution that is free for small teams.

  • Making a Website

    If you want somewhere to show off your game, you can make a website with any paid website builder.

    ... or with a basic understanding of HTML, you can modify existing HTML templates from designers like HTML5Up and host it on GitHub Pages for free.

    Kevin Stratvert on YouTube has a tutorial providing a basic understanding of HTML.
